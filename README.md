# Описание проекта
В ходе выполнения работы было создано консольное приложение **Bank**, содержащее класс "BankAccount", которы реализует работу банковского счета

Класс поддерживает следующие операции:
- Debit - снятие денежных средств со счета
- Credit - зачисление денежных средств на счет

Для тестирования приложения был создан отдельный проект **BankTests**, содержащий модульные тесты
# Документирование кода
Для документирования программного кода использовались XML-комментарии языка C#
Основные используемые теги:
- "<summary>" - описание класса или метода
- "<param>" - описание параметров
- "<returns>" - описание возвращаемого значения
- "<exception>" - описание возможных исключений
Документирование  позволяет улучшить читаемость кода
